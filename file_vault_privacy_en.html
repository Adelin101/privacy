File Vault Privacy Policy

Last Updated: 2025-01-01

1. Introduction

File Vault (“the app”) respects and protects your privacy. This Privacy Policy explains how we collect, use, and safeguard your data.

2. Information We Collect

  • Device Information: Network status, crash logs, usage statistics.
  • File Data: Files you choose for encryption, decryption, preview, or exporting.
  • Recovery Data: Mnemonic recovery is processed locally and never uploaded.

3. Permissions We Use

  • INTERNET / ACCESS_NETWORK_STATE: For connectivity and network type detection.
  • FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC: For required foreground tasks.
  • READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE (Android ≤ 32): For file operations.
  • WAKE_LOCK: To keep operations running.
  • POST_NOTIFICATIONS: To deliver notifications.
  • REQUEST_INSTALL_PACKAGES: For APK installation.
  • USB permissions: For accessing USB storage devices.

4. Third‑Party Services

  • Firebase Analytics (usage statistics)
  • Firebase Crashlytics (crash reporting)
  • Google Privacy Policy: https://policies.google.com/privacy

5. How We Use Your Information

  • Provide core features: encrypt, decrypt, preview, export.
  • Improve performance and user experience.
  • Analyze crashes and fix issues.

6. Data Storage & Security

All encryption‑related processing is done locally. The app does not upload your files, mnemonic phrases, or sensitive data.

7. Your Rights

  • Access or delete local data.
  • Withdraw permissions (may affect functionality).

8. Contact Us

If you have privacy concerns, please contact us via the email provided within the app.
