File Vault Privacy Policy

Last Updated: December 2, 2025

1. Introduction

File Vault (“the app”) is committed to protecting your privacy and data security. This Privacy Policy explains in detail what information we collect, how we use it, and your rights as a user.

2. Information We Collect

  • Device Information: Device model, OS version, network status, app version, used to enhance performance and compatibility.
  • Usage Statistics: Collected via Firebase Analytics for anonymous app usage and crash reporting, used only to improve app experience.
  • File Data: Files added to the encrypted Vault are processed locally for encryption, decryption, preview, and export. No data is uploaded.
  • Recovery Data: Mnemonic phrases are processed locally and never transmitted or stored externally.

3. Permissions Usage

  • INTERNET / ACCESS_NETWORK_STATE: For network detection and anonymous statistics.
  • FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC: For long-running tasks like encryption/decryption.
  • READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE (Android ≤ 32): For accessing files for encryption and export.
  • WAKE_LOCK: Ensures long tasks are not interrupted.
  • POST_NOTIFICATIONS: Notify users when tasks complete.
  • REQUEST_INSTALL_PACKAGES: For APK installation functionality.
  • USB permissions: To access user-authorized external USB devices.

4. Third-Party Services

  • Firebase Analytics: Anonymous usage statistics.
  • Firebase Crashlytics: Crash reporting for stability improvement.
  • Google Privacy Policy: https://policies.google.com/privacy

5. How We Use Your Information

  • Provide core features: encryption, decryption, preview, and export.
  • Analyze crashes and optimize performance.
  • Ensure local file security; no sensitive information is uploaded.

6. Data Storage & Security

All files and mnemonic phrases are stored and processed locally on your device. The app does not transmit any sensitive information. System security mechanisms prevent unauthorized access.

7. User Rights

  • Access or delete local data.
  • Withdraw permissions (may affect some functionality).
  • Disable network access or third-party analytics.

8. Minor Protection

The app does not knowingly collect personal information from minors. If you are under 18, please use the app under parental guidance.

9. Policy Updates

We may update this Privacy Policy to comply with legal requirements, functional improvements, or security needs. Updates will be posted in the app or on the website, with user notification on first launch.
