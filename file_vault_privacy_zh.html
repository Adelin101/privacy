File Vault 隐私政策

最后更新日期：2025 年 12 月 02 日

1. 简介

File Vault（以下简称“本应用”）致力于保护您的隐私与数据安全。本隐私政策详细说明我们收集的信息类型、使用目的以及用户权利，帮助您充分了解我们的数据处理方式。

2. 我们收集的信息

  • 设备信息：设备型号、系统版本、网络状态、应用版本，用于提升性能与兼容性。
  • 使用统计：通过 Firebase Analytics 收集应用功能使用情况和崩溃日志，仅用于优化应用体验和修复问题。
  • 文件信息：用户添加到加密库的文件仅在本地处理，用于加密、解密、预览和导出。不会上传到服务器。
  • 助记词信息：仅用于本地恢复 Vault，不会存储或上传。

3. 权限使用说明

  • INTERNET / ACCESS_NETWORK_STATE：用于统计与崩溃分析，以及检测网络环境。
  • FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC：用于后台加密、解密等长耗时任务。
  • READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE（Android ≤ 32）：用于访问用户文件进行加密和导出。
  • WAKE_LOCK：保证长时间任务不会被系统中断。
  • POST_NOTIFICATIONS：用于提示任务完成状态。
  • REQUEST_INSTALL_PACKAGES：在安装本地 APK 功能中使用。
  • USB 权限：用于访问用户授权的外接 USB 设备。

4. 第三方服务

  • Firebase Analytics：用于匿名统计应用使用情况。
  • Firebase Crashlytics：用于捕获崩溃信息，提高应用稳定性。
  • Google 隐私政策：https://policies.google.com/privacy

5. 信息使用方式

  • 提供核心功能，包括加密、解密、预览和导出文件。
  • 分析崩溃和性能，优化应用体验。
  • 保护用户文件安全，确保数据不会泄露或上传。

6. 数据存储与安全

所有文件和助记词均在用户设备本地存储和处理，本应用不上传任何敏感信息。我们使用系统安全机制防止未授权访问。

7. 用户权利

  • 访问或删除本地数据。
  • 撤回权限（可能会影响某些功能）。
  • 选择不使用网络访问或第三方统计功能。

8. 未成年人保护

本应用不会故意收集未成年人的个人信息。如您为未成年人，请在监护人指导下使用。

9. 政策更新

我们可能会根据法律、功能更新或安全需求更新本隐私政策。更新后会在应用内或官网公布，并在应用首次启动时提示用户阅读。
