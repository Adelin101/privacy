File Vault 隐私政策

最后更新日期：2025-01-01

1. 简介

File Vault（以下简称“本应用”）尊重并保护用户的隐私。本隐私政策旨在说明我们如何收集、使用和保护用户的数据。

2. 我们收集的信息

  • 设备信息：包括网络状态、应用崩溃日志、使用统计。
  • 文件相关数据：用户选择加密、解密、导出等操作的文件。
  • 助记词恢复数据：仅在设备本地处理，不上传至服务器。

3. 使用的权限

  • INTERNET / ACCESS_NETWORK_STATE：用于联网和区分网络类型。
  • FOREGROUND_SERVICE / FOREGROUND_SERVICE_DATA_SYNC：用于必要的前台服务。
  • READ_EXTERNAL_STORAGE / WRITE_EXTERNAL_STORAGE（Android 32 以下）：用于读取与写入加解密文件。
  • WAKE_LOCK：用于保持任务运行。
  • POST_NOTIFICATIONS：用于发送本地通知。
  • REQUEST_INSTALL_PACKAGES：用于 APK 安装流程。
  • USB 权限：用于访问外接 USB 设备。

4. 第三方服务

  • Firebase Analytics（统计）
  • Firebase Crashlytics（错误分析）
  • Google 隐私政策：https://policies.google.com/privacy

5. 我们如何使用信息

  • 提供核心功能：加密、解密、预览、导出文件。
  • 改进用户体验与性能优化。
  • 诊断问题与修复崩溃。

6. 数据存储与安全

所有加密相关数据均在用户本地设备处理。本应用不上传用户的文件、助记词或任何敏感数据。

7. 用户权利

  • 访问、删除本地数据。
  • 撤回权限（可能影响功能）。

8. 联系我们

如有隐私相关问题，请通过应用内提供的联系邮箱联系。
